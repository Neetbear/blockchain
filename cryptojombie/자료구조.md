배열 -> 장점 : 다루기 쉽다, index가 존재한다(메모리 주소값)
    -> 단점 : 배열 중간 위치의 처리가 어렵다 -> 새로운 배열만들어서 처리하니까 메모리적으로 비효율

Linked List : node (값 + next node 주소값)
    구현 알고리즘 복잡하다 
    index로 찾아가는 방법이 불가능하다
    중간 애들 처리가 쉽다 ( 주소값 변경해서 연결을 바꾸면 된다 ) 

tree : root에서 leef까지 연결된 node 구조
